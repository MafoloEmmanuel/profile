Bootcamp Unit Testing

In this project I learned how to test my bootcamp functions. I learned how to use ChaiJS asserts to check function results. Write code in the unit test that is calling the function under test with different parameters checking if the function behaves correctly. If it behaves correctly the results will be correct, but if not it will return a failure/error.

DOM Widgets Intro

I learned that the DOM is about the browser, what people can see and do with an application. So we use the DOM to gather and display information from end users. We have many DOM elements such as textboxes, radio buttons, checkboxes, div elements and other html tags. I learned about how the DOM loads and how to build a DOM widget.

Unit Testing DOM Widgets

In this project I learned that a factory function is a functions that can return object literals and other functions. I learned how to keep the application logic and the DOM code separate to make the functions testable. To focus purely on testing the function logic and not the DOM code. I learned how to use the factory functions to test my logic. To test all the business logic of my DOM widgets using Mocha.
