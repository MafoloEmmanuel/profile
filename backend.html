Settings Bill WebApp

The project shows the total for the bill of calls and smses made. You set your own cost for the call and sms, and also the warning and critical levels. Select the call or sms radio button to make a call/sms. Totals updates when the ADD button is pressed. If the total cost exceed the warning level show the total cost in orange, if it exceeds the critical level show it in red and prevent any new costs from being added.

Greetings WebApp

In this project I learned How to create a webpage in which a user puts a name in a textbox and the the select a language in which they want to greet the name in. So the user has to choose between three languages. After the user put a name and select a language , a greetings message has to be displayed saying for example "Hello, Emmanuel".
